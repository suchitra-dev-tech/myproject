GitHub Collaboration For 6th B

Admin & Developer Workflow

👨‍💼 Admin Role

  • Created Repository
  • Added Collaborators
  • Review Pull Requests
  • Approve & Merge Code

👩‍💻 Developer Role

  • Clone Repository
  • Create New Branch
  • Add Feature (Edit this file)
  • Push Changes
  • Create Pull Request

🔁 Collaboration Flow

  1. Admin creates repository
  2. Developers clone repository
  3. Developers create feature branch
  4. Developers make changes
  5. Developers raise Pull Request
  6. Admin reviews and merges
